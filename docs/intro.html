1  Week1 - Introduction of RS

1.1 Beginning

在开始这门课之前，我需要首先了解这一课程的目的，是介绍远程感知的基本概念和应用，并且通过RS这门课，我们即将学习到的内容包括：

  • 远程感知的定义、分类和平台

  • 电磁波与地球表面的相互作用

  • 远程感知数据的四个分辨率：空间、光谱、时间和辐射

  • 远程感知数据的格式、选择和限制

  • 远程感知数据的实际案例和分析方法

这很容易让我联想到我过去本科期间自学到的RS的基础概念，我相信这门课不仅能帮助我温习过去的知识，还能充分补充其他的知识。

1.2 Summary of this class

  • 远程感知城市和环境：这是一门介绍远程感知基本原理和应用的课程，主要关注城市和环境问题。

  • 远程感知的类型和原理：远程感知分为主动和被动两种，根据是否有自身的能源发射器。远程感知数据受到电磁波与大气和地表的相互作用的影响，需要进行校正和处理。

  • 远程感知数据的格式和分辨率：远程感知数据通常是栅格数据，有不同的存储格式和组织方式。远程感知数据的质量和应用受到空间、光谱、时间和辐射分辨率的制约。

1.3 遥感的基本概念与原理

1.3.1 被动传感器与主动传感器的区别

被动传感器：使用现有的能量，不发射任何东西，检测来自太阳的反射能量，例如人眼、相机、卫星传感器。

主动传感器：有自己的能量源，主动发射电磁波，然后等待接收，例如雷达、X射线、LiDAR。

被动和主动传感器的区别：被动传感器受大气散射的影响，需要在光照条件良好的时候工作，而主动传感器可以穿透云层、火山灰和大气条件，可以在夜间工作。

主动传感器的例子：合成孔径雷达（SAR），可以“看穿云层”，并且具有极化特性，可以根据表面的粗糙度、形状、方向、湿度、盐度、密度等反映出不同的电磁波。
